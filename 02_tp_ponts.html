🌉 TP — Ingénieur des ponts

5 missions progressives. Chaque mission réussie déverrouille la suivante. À la fin : le code du jeu Bâtisseur Fou !

⭐ 0 pts

1Reconnaître les efforts

Pour chaque situation, indique quel effort subit la pièce en rouge. (1 pt par bonne réponse du premier coup)

2Simulateur de poutre

Un camion de 40 t doit traverser. Règle la portée, le matériau et la hauteur de poutre pour que la flèche (déformation) reste sous la limite. 🎯 Mission : réussir une traversée avec une portée d'au moins 30 m.
Flèche maximale autorisée = portée ÷ 200.
Astuce d'ingénieur : doubler la hauteur divise la flèche par 8 !

3Rigidifier la tour de grue

Cette tour de grue est faite de 3 caissons carrés articulés : au moindre coup de vent, elle se déforme ! Clique sur un caisson pour y ajouter une barre diagonale (3 barres disponibles), puis lance le test du vent.
Barres diagonales posées : 0 / 3
Rappel du cours : la seule figure indéformable est le triangle.

4Franchir la gorge du Diable

Une gorge de 120 m de large, très profonde (les piles coûtent cher !). Budget : 100 k€. Choisis le type de pont, le matériau et le nombre de piles, puis teste. 🎯 Il existe plusieurs solutions… trouve la plus économique pour gagner 3 étoiles !
Coût total : 16 k€ / 100 k€

Dernière ligne droite : au moins 3 bonnes réponses sur 4 pour obtenir le code du jeu.
