Adresses IP & DNS : l'annuaire d'Internet

Chaque machine a une adresse IP (ex : 172.217.20.78) — son numéro dans le réseau. Mais qui retient des numéros ? Toi tu tapes un nom de domaine…

estic-saintdizier.fr (nom pour les humains) ⟷ 92.243.16.108 (IP pour les machines)
Le serveur DNS fait la traduction, comme un annuaire téléphonique.

✅ À retenir

  • Sur Internet, les données voyagent en paquets indépendants, aiguillés de routeur en routeur, puis réassemblés à l'arrivée.
  • Si une route tombe en panne, les paquets passent par un autre chemin : le réseau maillé rend Internet résistant.
  • Le DNS traduit les noms de domaine (estic.fr) en adresses IP — c'est l'annuaire d'Internet.
  • Temps de téléchargement = taille du fichier ÷ débit (ex : 600 Mo à 50 Mo/s → 12 s).
  • Le chiffrement (HTTPS 🔒) rend les messages illisibles pour les espions ; mot de passe fort + vigilance anti-phishing = tes meilleures défenses.
